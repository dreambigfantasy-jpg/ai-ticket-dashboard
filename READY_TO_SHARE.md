## 🚀 In 5 Minutes - Push Your Code to GitHub

### 1. Create GitHub Repo (2 min)
Go to https://github.com/new
- Name it: `ai-ticket-dashboard`
- Click "Create repository"
- **Copy the URL** (you'll need it next)

### 2. Push Your Code (3 min)
Open PowerShell in your project folder and paste:

```bash
git config --global user.name "Your Name"
git config --global user.email "your.email@example.com"

git add .
git commit -m "Initial commit: AI Ticket Dashboard"
git remote add origin https://github.com/YOUR-USERNAME/ai-ticket-dashboard.git
git push -u origin main
```

**Replace:**
- `Your Name` → Your actual name
- `your.email@example.com` → Your email
- `YOUR-USERNAME` → Your GitHub username

✅ **Done!** Check GitHub.com to see your code uploaded.

## 👥 Anyone Else Can Run It In 3 Commands

Share your GitHub URL: `https://github.com/YOUR-USERNAME/ai-ticket-dashboard`

Then they run:

```bash
# 1. Clone (download)
git clone https://github.com/YOUR-USERNAME/ai-ticket-dashboard.git
cd ai-ticket-dashboard

# 2. Install dependencies (waits 2-3 mins)
npm install

# 3. Start server
npm start
```

Open browser: http://localhost:3000
